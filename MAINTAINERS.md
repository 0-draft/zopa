# Maintainers

Maintainers have commit access and are responsible for review,
release, and roadmap decisions. See [GOVERNANCE.md](GOVERNANCE.md)
for how decisions are made.

| Name    | GitHub                                 | Areas                   |
| ------- | -------------------------------------- | ----------------------- |
| kanywst | [@kanywst](https://github.com/kanywst) | core, proxy-wasm, build |

## Becoming a maintainer

Open a pull request that adds your row to the table above. Approval
requires lazy consensus among existing maintainers (see
[GOVERNANCE.md](GOVERNANCE.md)). Sustained contribution and good
review judgment are the bar; specific commit counts are not required.

## Stepping down

Open a pull request that removes your row. No justification required.
